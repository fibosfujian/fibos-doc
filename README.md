FIBOS 在线文档
==========

> FIBOS是开发者友好的区块链生态系统。基于JavaScript 开发 + BANCOR 协议智能通证 + 开发者服务，FIBOS 平台实现了快速开发、快速部署和稳定且流动的通证体系，帮助开发者一步进入区块链时代。

本书将简单如何快速上手使用FIBOS。
